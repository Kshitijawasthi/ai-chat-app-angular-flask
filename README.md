# AI Chat App 🚀

A full-stack AI chat application built using:

- Angular (Frontend)
- Flask (Backend)
- Gemini API (AI)

## Features
- Real-time chat UI
- AI-generated responses
- Clean UI with auto-scroll

## Run Locally

### Backend
cd backend-app
pip install -r requirements.txt
python app.py

### Frontend
cd frontend-app
npm install
ng serve
